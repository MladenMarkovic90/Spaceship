You will need to download Freeglut3 from: http://freeglut.sourceforge.net/

Make sure the image files are in same folder as the game file

Controls:

R -> Start/Continue
T -> Pause
SPACE -> Attack
W -> move up
A -> move left
S -> move down
D -> move right
ESC -> exit
